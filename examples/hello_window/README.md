# hello_window

[![License](https://img.shields.io/badge/license-MIT%2FApache-blue.svg)](https://github.com/alan13367/Oxide-Core#license)

Basic window creation example for Oxide Core.

## Overview

This example demonstrates the fundamentals of creating an Oxide Core application with a 3D scene and FPS-style camera controls.

## Running

```bash
cargo run -p hello_window
```
